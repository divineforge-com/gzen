# gZen.io Branding Guidelines

This skill defines the rules for gZen.io branding, naming, typography, color palette, and language options across the monorepo.

## Naming & Voice
- Use **gZen.io** or **gZen** (lowercase g, capital Z). Never "GrowZen" or "GZen".
- Keep descriptions focused on:
  - **Advocating energy flow** (身心能量与系统架构的顺畅流动)
  - **Right tool use** (克制用器, Clarity before tools)
  - **Real impact** (长期主义, 真实影响与善意)

## Visual Design System
- **Background**: Warm cream (`#fffaf4`), never cold gray or pure white.
- **Ink**: Earthy dark brown (`#2f2118`), never pure black.
- **Saffron**: Warm saffron/monk's robe amber (`#d97845`).
- **Jade**: Warm jade green (`#2d6b4f`) for gZen Ki.
- **Typography**: CJK Cursive/Serif first (Noto Serif SC for titles, Noto Sans SC for body). Line height `≥ 1.8` for readability.

## Multilingual Menu Order
- Always sort translation menus with **English first**, then **Chinese**, then **Japanese**.
  - `en` (Weight: 1)
  - `zh` (Weight: 2)
  - `ja` (Weight: 3)

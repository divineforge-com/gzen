## 什么是八段锦

八段锦（Bā Duàn Jǐn）是中国最古老、流传最广的健身气功之一。「八段」指八节动作，「锦」意为精美华丽——将这八节动作比作光彩夺目的锦缎。

起源可追溯至**北宋时期**（约公元12世纪），相传由八仙之一钟离权所传，亦有说岳飞所创。清代宫廷中广泛习练，现有坐式与立式两种版本。本文介绍**立式八段锦**。

整套练习约 **10–15分钟**，无需器械，约2×2米空间即可完成。

## 练习前准备

**时间**：早晨起床后（辰时，7–9时，胃经当令）或傍晚为佳  
**场所**：通风良好，空气清新  
**着装**：宽松舒适，平底软鞋或赤足  
**呼吸**：全程腹式深呼吸，鼻吸鼻呼，口唇轻合  
**心态**：排除杂念，意守丹田（下腹部）

## 第一式：双手托天理三焦

**动作要领**

1. 两脚平行，与肩同宽，自然站立
2. 十指交叉，掌心向上，缓缓上托至头顶
3. 手臂伸直，仰视手背，同时缓缓吸气
4. 稍作停顿，感受脊椎纵向拉伸
5. 呼气，双手松开，由两侧缓缓落下还原
6. 重复 **6–8次**

**呼吸配合**：上托时吸气，还原时呼气

**中医功效**

| 靶向部位 | 功效 |
|----------|------|
| 三焦经 | 疏通上、中、下三焦，调理全身气机 |
| 脊椎 | 纵向拉伸，调整椎间盘 |
| 肩颈 | 舒展肩颈肌肉，缓解颈椎僵硬 |

> 「三焦」是中医特有概念，掌管全身水液代谢与气的升降。此式能疏通三焦，是全套的总调节动作。

## 第二式：左右开弓似射雕

**动作要领**

1. 双脚分开约两倍肩宽，屈膝下蹲成**马步**
2. 左手成「八字掌」向左推出，食指朝上
3. 右手成拉弓状，右肘向右拉开，目视左食指尖
4. 左右交替进行，各 **3–4次**
5. 还原时双腿伸直，回到自然站立

**呼吸配合**：下蹲展臂时吸气，还原时呼气

**中医功效**

| 靶向部位 | 功效 |
|----------|------|
| 肺经 | 扩胸展肩，增大肺活量 |
| 心包经 | 开胸理气，调和心气 |
| 肝经 | 目视远方，疏肝明目 |

> 马步是中医武术中培元固本的基本桩法，能强健腰腿，使气血下行，防止上焦火旺。

## 第三式：调理脾胃须单举

**动作要领**

1. 自然站立，右手翻掌上举过头，掌心向上
2. 同时左手下按，掌心向下，两手形成对向撑力
3. 保持 2–3秒，感受躯干侧向拉伸
4. 还原后换左手上举，右手下按
5. 左右各 **3–4次**

**呼吸配合**：举手时吸气，还原时呼气

**中医功效**

| 靶向部位 | 功效 |
|----------|------|
| 脾经 | 拉伸脾经，健运脾气 |
| 胃经 | 按摩腹腔脏器，促进胃肠蠕动 |
| 肝胆 | 侧身伸展疏通肝胆之气 |

> 中焦脾胃的「升清降浊」功能与此式一升一降的动作方向完全契合，是脾胃虚弱者的核心练习。

## 第四式：五劳七伤往后瞧

**动作要领**

1. 自然站立，两臂自然垂于体侧
2. 两手心向外旋转，肩胛骨微微内收
3. 头颈缓缓向左转至极限（不可超出舒适范围）
4. 保持 2秒，目光向左后方远望
5. 还原后向右侧重复
6. 左右各 **3–4次**

**呼吸配合**：转头时吸气，还原时呼气

**中医功效**

| 靶向部位 | 功效 |
|----------|------|
| 督脉 | 刺激颈椎，振奋阳气 |
| 膀胱经 | 拉伸背部膀胱经，消除疲劳 |
| 眼目 | 运目活络，改善眼部气血 |

> 「五劳」指心、肝、脾、肺、肾五脏之劳损；「七伤」指各类内外损伤。此式以颈椎旋转激活督脉，对慢性疲劳综合症有良好效果。

## 第五式：摇头摆尾去心火

**动作要领**

1. 双脚分开约两倍肩宽，屈膝下蹲成**马步**
2. 双手扶膝，掌心向内
3. 上身向左前方俯身，头向左摇，尾骨（臀部）向右摆
4. 以脊椎为轴，做弧形摇摆
5. 还原后换方向，左右各 **3–4次**

**呼吸配合**：摇摆时呼气，还原时吸气

**中医功效**

| 靶向部位 | 功效 |
|----------|------|
| 心经 | 俯身活动脊柱，清泄心火 |
| 肾经 | 腰部运动培补肾气，水克火 |
| 督任二脉 | 脊椎摇摆激活督脉，疏通全身阳气 |

> 「心火」常见表现：口舌生疮、心烦失眠、小便短赤。此式以肾水制心火，是夏季和情绪紧张时的重要练习。

## 第六式：两手攀足固肾腰

**动作要领**

1. 自然站立，双腿伸直
2. 双手从腰部沿脊柱两侧向上推至肩胛（摩擦命门穴区域）
3. 向前深度俯身，双手顺腿向下触及脚踝或地面（量力而为）
4. 还原，以腰带身直立
5. 重复 **6–8次**，腿膝保持伸直

**注意**：腰椎间盘突出者请减小前屈幅度，以舒适为准。

**呼吸配合**：俯身时呼气，直立时吸气

**中医功效**

| 靶向部位 | 功效 |
|----------|------|
| 肾经 | 按摩命门、肾俞穴，温补肾阳 |
| 膀胱经 | 拉伸整条脊背膀胱经 |
| 肝经 | 拉伸腿部肝经，柔肝滋阴 |

> 命门穴是督脉上的重要穴位，位于第二腰椎棘突下，是肾阳的根本。向上推摩此区域有直接温肾作用。

## 第七式：攥拳怒目增气力

**动作要领**

1. 马步站立，两手握拳置于腰侧（拳心向上）
2. 左拳缓缓向前冲出，同时右拳向右后方反向用力
3. 出拳时「怒目」——双目圆睁，凝神聚气
4. 还原后换右拳出击，左右各 **3–4次**
5. 每次出拳配合呼气，短促有力

**呼吸配合**：出拳时短促呼气，收拳时吸气

**中医功效**

| 靶向部位 | 功效 |
|----------|------|
| 肝经 | 肝开窍于目，怒目圆睁激活肝经气血 |
| 宗筋 | 马步强化腰腿筋力，培固元气 |
| 全身气血 | 发劲出拳推动气血在经络中涌动 |

> 「怒目」不是发怒，而是通过眼睛的「威猛之神」激发肝经阳气，增强正气。此式是八段锦中气感最明显的动作之一。

## 第八式：背后七颠百病消

**动作要领**

1. 双脚并拢，自然站立，全身放松
2. 脚跟缓缓离地，踮起脚尖，同时头顶向上虚领
3. 保持 1–2秒，全身保持挺拔
4. 脚跟轻轻落地，略有震动感（非猛力顿足）
5. 重复 **7次**（「七颠」之意）

**呼吸配合**：踮起时吸气，落下时呼气

**中医功效**

| 靶向部位 | 功效 |
|----------|------|
| 督脉 | 脚跟震动沿脊柱传导，激活督脉 |
| 全身经络 | 震动按摩五脏六腑，疏通经络 |
| 小肠·肾经 | 踮脚锻炼足跟，通达肾经与小肠经 |

> 此式是全套的「收功」动作——通过轻微震动让全身气血归于平稳，犹如整理一根刚抖开的锦缎，让它重归整齐。

## 整套时间安排

| 阶段 | 内容 | 时间 |
|------|------|------|
| 热身 | 站桩调息、简单拉伸 | 2–3分钟 |
| 一至四式 | 上焦·心肺·脊椎 | 5–6分钟 |
| 五至八式 | 中下焦·脾肾·总收功 | 5–6分钟 |
| 收功 | 搓手浴面、站桩静息 | 2–3分钟 |
| **合计** | | **约15分钟** |

## 每日练习建议

- **初学者**：每天1套，专注动作准确，不急求次数
- **进阶者**：早晚各1套，或每套重复2–3遍
- **体虚者**：可先练坐式八段锦，逐步过渡到立式

练习完毕后，双手互搓至温热，然后轻轻浴面（搓脸）——这是中医传统的收功方式，可将练功产生的气引至面部，滋润皮肤。

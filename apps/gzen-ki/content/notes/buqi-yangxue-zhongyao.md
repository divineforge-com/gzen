## 补气类中药

### 黄芪（おうぎ / Huáng Qí）

> 《神农本草经》列为上品

**药材来源**：豆科植物蒙古黄芪或膜荚黄芪的干燥根

| 属性 | 内容 |
|------|------|
| **性味** | 甘，微温 |
| **归经** | 肺、脾经 |
| **功效** | 补气升阳、固表止汗、利水消肿、托毒生肌 |
| **主治** | 脾肺气虚、气虚发热、气不摄血、表虚自汗 |
| **用量** | 9-30g（大剂量可用至120g）|

**现代药理**：
- 增强免疫功能（提升 NK 细胞、T 细胞活性）
- 保护心血管（扩张冠状动脉、改善心肌缺血）
- 抗氧化、抗肿瘤辅助作用
- 调节血糖（黄芪多糖有降糖作用）

**经典方剂**：补中益气汤、玉屏风散、黄芪建中汤

---

### 人参（にんじん / Rén Shēn）

**药材来源**：五加科植物人参的干燥根

| 属性 | 内容 |
|------|------|
| **性味** | 甘、微苦，微温 |
| **归经** | 脾、肺、心、肾经 |
| **功效** | 大补元气、复脉固脱、补脾益肺、生津养血、安神益智 |
| **主治** | 体虚欲脱、肢冷脉微、脾虚食少、肺虚咳嗽、津伤口渴 |
| **用量** | 3-9g（急救回阳可用15-30g）|

**注意事项**：
- 实热证、正气未虚者禁用
- 反藜芦，畏五灵脂，恶皂荚
- 服药期间忌食萝卜（影响补气效果）

---

### 党参（とうじん / Dǎng Shēn）

**特点**：补气效力较人参温和，价格实惠，适合日常调补

| 属性 | 内容 |
|------|------|
| **性味** | 甘，平 |
| **归经** | 脾、肺经 |
| **功效** | 健脾益肺、养血生津 |
| **主治** | 脾肺气虚、食少倦怠、咳嗽虚喘、气血不足 |
| **用量** | 9-30g |

| 属性 | 内容 |

## 补血类中药

### 当归（とうき / Dāng Guī）

> 「十方九归」——当归是中医最常用的补血活血药

**药材来源**：伞形科植物当归的干燥根

| 属性 | 内容 |
|------|------|
| **性味** | 甘、辛，温 |
| **归经** | 肝、心、脾经 |
| **功效** | 补血活血、调经止痛、润肠通便 |
| **主治** | 血虚萎黄、月经不调、闭经痛经、虚寒腹痛、血虚便秘 |
| **用量** | 6-12g |

**当归各部位功效**：
- **当归头**：止血
- **当归身**：补血养血
- **当归尾**：活血化瘀
- **全当归**：补血活血兼顾

### 阿胶（あきょう / Ē Jiāo）

**药材来源**：马科动物驴的皮经漂洗熬制成的固体胶

| 属性 | 内容 |
|------|------|
| **性味** | 甘，平 |
| **归经** | 肺、肝、肾经 |
| **功效** | 补血滋阴、润燥、止血 |
| **主治** | 血虚、吐血、尿血、便血、崩漏、阴虚咳嗽、虚烦失眠 |
| **用量** | 3-9g（烊化兑服）|

**食用方法**：
- 阿胶膏：阿胶 250g + 核桃 + 红枣 + 黑芝麻 + 冰糖，熬制成膏
- 阿胶粥：阿胶溶化后加入熟粥中

## 气血双补经典方剂

### 四物汤（補血の基本方 / Four Substance Decoction）

**组成**：熟地黄 12g、白芍 12g、当归 10g、川芎 8g

**功效**：补血和血、调经化瘀  
**适应证**：血虚血滞、月经不调、经期腹痛、面色苍白

---

### 八珍汤（八珍湯 / Eight Treasure Decoction）

**组成**：四君子汤 + 四物汤（气血双补）

**功效**：气血双补  
**适应证**：气血两虚、面色苍白或萎黄、头晕目眩、心悸气短、食少体倦

---

### 归脾汤（帰脾湯 / Restore the Spleen Decoction）

**组成**：党参、黄芪、白术、茯苓、当归、远志、酸枣仁、木香、桂圆肉、大枣、甘草、生姜

**功效**：健脾益气、养血安神  
**适应证**：心脾两虚、气血不足、心悸健忘、失眠多梦、崩漏便血

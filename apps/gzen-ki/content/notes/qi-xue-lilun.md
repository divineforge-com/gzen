## 什么是「气」？

**气**（きき / Qì）是中医最基础的概念之一，指构成人体和维持生命活动的最基本物质与功能。气既是物质，又是功能，贯穿于人体一切生理活动之中。

### 气的主要功能（気の主な機能 / Functions of Qi）

| 功能 | 说明 |
|------|------|
| **推动作用** | 推动血液运行、脏腑功能活动与机体生长发育 |
| **温煦作用** | 维持人体正常体温，温暖脏腑、经络 |
| **防御作用** | 护卫肌表，抵御外邪入侵（即「正气存内，邪不可干」） |
| **固摄作用** | 固定脏腑位置，统摄血液、汗液、尿液等不外泄 |
| **气化作用** | 促进精、血、津液的相互转化与代谢 |

### 气的分类

```
元气（先天之气）
  ├── 宗气（积于胸中，司呼吸）
  ├── 营气（行于脉中，化生血液）
  └── 卫气（行于脉外，护卫肌表）
```

## 什么是「血」？

**血**（けつ / Xuè）是循行于脉中、富含营养的红色液态物质，主要功能是**滋养**和**濡润**全身脏腑组织。

### 血的生成

> 「中焦受气取汁，变化而赤，是谓血。」——《灵枢·决气》

血的生成来源：
1. **脾胃运化**：水谷精微经脾胃运化，上输于心肺，变化为血
2. **肾精转化**：肾中精气可化生血液（精血同源）
3. **肺气参与**：肺朝百脉，参与血的生成与输布

### 血的功能

- **滋养濡润**：营养全身，使皮肤红润、毛发光泽、筋骨有力
- **化神养神**：血是神志活动的物质基础，「血气者，人之神」
- **维持体温**：血液循环保持正常体温

## 气与血的关系

**气血相互依存，不可分割。**

### 气为血之帅

- 气能生血：气的运动推动血液生成
- 气能行血：气的推动作用维持血液正常运行
- 气能摄血：气的固摄作用防止血液溢出脉外

### 血为气之母

- 血能载气：气附于血中，靠血液运载至全身
- 血能养气：血液为气的活动提供物质基础

> 「气行则血行，气滞则血瘀。」

## 气血失调的常见证型

### 气虚证（気虚 / Qi Deficiency）

**症状**：疲乏无力、气短懒言、自汗、面色㿠白、舌淡苔白、脉虚弱

**调理方向**：补气为主  
**常用中药**：黄芪、党参、人参、白术、山药

---

### 气滞证（気滞 / Qi Stagnation）

**症状**：胸胁胀痛、情绪抑郁、嗳气频繁、腹胀、舌苔薄、脉弦

**调理方向**：疏肝理气  
**常用中药**：柴胡、香附、陈皮、枳壳、郁金

---

### 血虚证（血虚 / Blood Deficiency）

**症状**：面色苍白或萎黄、唇甲无华、头晕目眩、心悸失眠、手足麻木、舌淡、脉细

**调理方向**：补血养血  
**常用中药**：当归、熟地黄、白芍、阿胶、龙眼肉

---

### 血瘀证（血瘀 / Blood Stasis）

**症状**：刺痛（固定不移）、皮肤紫暗或瘀斑、口唇紫绀、舌质紫暗有瘀点、脉涩

**调理方向**：活血化瘀  
**常用中药**：丹参、桃仁、红花、川芎、益母草

---

## 气血调养的日常建议

1. **规律作息**：「子时一阳生」，23:00前入睡，护养肝血
2. **适度运动**：太极拳、八段锦——动而不过，促进气血流通
3. **情志调节**：「百病生于气」，保持心情平和，避免长期压力
4. **饮食调理**：温热饮食，避免生冷，多食红枣、桂圆、阿胶等补血之品
5. **艾灸补气**：灸足三里、关元、气海等穴位，培补元气

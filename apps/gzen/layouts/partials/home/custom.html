Stack

This site is a living demonstration of Go engineering principles. Built with:

  • Hugo — the world's fastest static site generator, itself written in Go. Zero runtime dependencies, instant CDN delivery.
  • gzen-tool — a custom Go CLI for parallel multi-site builds, using Goroutines to compile all sub-sites in under 200ms.
  • Cloudflare Pages — global edge deployment with no servers to manage.
  • JetBrains Mono — code-first typography that communicates precision and craft.

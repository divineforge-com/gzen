{{ i18n "tech_stack_title" | default "极简与高效的技术架构" }}

本站及生态项目以克制用器为工程实践，展示如何使用高性能、零冗余的开源工具链构建轻量级云原生系统：

  • 高性能静态引擎：全部网站由 Go 语言编写的 Hugo 静态生成，零客户端运行时，通过 Cloudflare 全球 CDN 实现瞬时响应。
  • Go 语言开发脚手架：由我们自主开发的 gzen-tool Workspace CLI 工具接管整个 Monorepo（单仓多包）的构建，基于 Goroutine 协程实现多站点并行构建，编译仅需 200 余毫秒。
  • 轻量级 AI 自动化：后台集成了以 Go 编写、运行仅耗 <10MB 内存 的 AI 智能体 PicoClaw，借助 Gemini 2.5/3.5 模型自动根据农历节律完成内容分发与多语言翻译协同。

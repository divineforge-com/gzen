🆚 Value vs Growth Investing

📖 Value Investing

🎯Buy below intrinsic value
📊Low P/E, P/B ratios
💪Strong fundamentals
⏳Long-term horizon (5–10yr)
🏆Warren Buffett style
🔍DCF + Graham number analysis

🚀 Growth Investing

📈High revenue/earnings growth
💻Tech, AI, Biotech sectors
🔮Future potential over current value
⚡Higher risk, higher reward
🦅Cathie Wood / Peter Lynch style
📉Volatile, needs strong conviction

💡 Blend both — core value holdings (60%) + growth satellites (40%) for balanced wealth building

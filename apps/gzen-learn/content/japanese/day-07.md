## Grammar Pattern

**〜ばよかった**

Express wishes about the past using 〜ばよかった.

## Example Sentences

1. もっと<ruby>勉強<rt>べんきょう</rt></ruby>すればよかった。
   *I should have studied more.*
   　*（繁中：早知道應該多讀點書就好了。）*

2. <ruby>早<rt>はや</rt></ruby>く<ruby>起<rt>お</rt></ruby>きればよかった。
   *I should have woken up earlier.*
   　*（繁中：早知道早點起床就好了。）*

3. あのとき<ruby>正直<rt>しょうじき</rt></ruby>に<ruby>話<rt>はな</rt></ruby>せばよかった。
   *I should have spoken honestly back then.*
   　*（繁中：那時候如果能坦誠說出來就好了。）*

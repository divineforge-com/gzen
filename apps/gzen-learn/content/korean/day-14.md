> 🇹🇼 **為繁體中文使用者設計** — 本課學習韓語的地點助詞「에」和「에서」。中文用「在」表示所有地點關係，韓語則有兩個不同的助詞，分別用於「靜止狀態」和「進行動作」，一開始容易混淆，但有規律可循！

## 詞彙

| 韓語 | 羅馬拼音 | 詞性 | 繁中意思 | 例句 |
|------|---------|------|---------|------|
| 에 | e | 助詞 | 在……（靜態地點/方向）| 학교에 가요. (去學校。) |
| 에서 | e-seo | 助詞 | 在……（動作地點）/ 從…… | 도서관에서 공부해요. (在圖書館學習。) |
| 에서…까지 | e-seo…kka-ji | 助詞 | 從……到…… | 서울에서 부산까지 가요. (從首爾到釜山。) |
| 위 | wi | 名詞 | 上面 | 책상 위에 있어요. (在桌子上面。) |
| 아래 | a-rae | 名詞 | 下面 | 의자 아래에 있어요. (在椅子下面。) |
| 앞 | ap | 名詞 | 前面 | 학교 앞에 있어요. (在學校前面。) |
| 뒤 | dwi | 名詞 | 後面 | 집 뒤에 있어요. (在房子後面。) |
| 옆 | yeop | 名詞 | 旁邊 | 편의점 옆이에요. (在便利商店旁邊。) |
| 안 | an | 名詞 | 裡面 | 가방 안에 있어요. (在包包裡面。) |
| 밖 | bak | 名詞 | 外面 | 밖에 나가요. (出去外面。) |

## 文法

### 에 vs 에서 的核心差別

| | 에 | 에서 |
|--|--|--|
| 主要用途 | 存在地點、移動目的地 | 動作進行地點、出發點 |
| 搭配動詞類型 | 있다/없다/가다/오다 | 먹다/공부하다/일하다 等行為動詞 |
| 中文類比 | 在……（靜止）/ 到…… | 在……做某事 / 從…… |

**記憶口訣：**
- **에** = 「**去到**哪裡」或「**靜靜地在**哪裡」
- **에서** = 「**動作發生**在哪裡」或「**從**哪裡出發」

### 에 的使用情境

| 情境 | 例句 | 中文 |
|------|------|------|
| 存在地點（있다/없다）| 학교**에** 있어요 | 在學校 |
| 移動目的地（가다/오다）| 서울**에** 가요 | 去首爾 |
| 時間表達 | 세 시**에** 만나요 | 三點見面 |
| 方向 | 왼쪽**에** 있어요 | 在左邊 |

### 에서 的使用情境

| 情境 | 例句 | 中文 |
|------|------|------|
| 動作地點 | 도서관**에서** 공부해요 | 在圖書館學習 |
| 出發地點 | 서울**에서** 왔어요 | 從首爾來 |
| 出發點（到達：까지）| 여기**에서** 역까지 | 從這裡到車站 |

### 位置詞（方位名詞）

韓語的方位詞後面要加 에 來表示位置：

| 位置詞 | 韓語 | 例句 |
|-------|------|------|
| 上面 | 위（에）| 책 위에 있어요 |
| 下面 | 아래/밑（에）| 침대 밑에 있어요 |
| 前面 | 앞（에）| 집 앞에 있어요 |
| 後面 | 뒤（에）| 학교 뒤에 있어요 |
| 旁邊 | 옆（에）| 편의점 옆에 있어요 |
| 裡面 | 안（에）| 가방 안에 있어요 |
| 外面 | 밖（에）| 밖에 있어요 |
| 左邊 | 왼쪽（에）| 왼쪽에 있어요 |
| 右邊 | 오른쪽（에）| 오른쪽에 있어요 |

### 常用地點詞彙

|------|---------|------|

| 도서관 | do-seo-gwan | 圖書館 |
| 병원 | byeong-won | 醫院 |
| 편의점 | pyeon-ui-jeom | 便利商店 |

## 例句

1. **에 — 存在地點**
   화장실이 어디에 있어요?
   저기 오른쪽에 있어요.
   *廁所在哪裡？*
   *在那邊右側。*

2. **에서 — 動作地點**
   어디에서 한국어를 공부해요?
   카페에서 공부해요.
   *在哪裡學韓語？*
   *在咖啡廳學習。*

3. **에서…까지 — 從……到……**
   여기에서 명동까지 얼마나 걸려요?
   지하철로 약 이십 분이에요.
   *從這裡到明洞要多久？*
   *搭地鐵大約二十分鐘。*

4. **位置詞**
   열쇠가 어디에 있어요?
   가방 안에 있어요.
   *鑰匙在哪裡？*
   *在包包裡面。*

5. **출발지 에서 — 出發地**
   어디에서 왔어요?
   대만에서 왔어요.
   *從哪裡來？*
   *從台灣來的。*

## 閱讀練習

저는 매일 학교에 가요. 학교에서 수업을 들어요. 점심에는 학교 옆 식당에서 밥을 먹어요. 수업 후에는 도서관에서 공부해요. 저녁에는 집에서 쉬어요. 집은 지하철역 앞에 있어요. 아주 편해요.

**翻譯：**
我每天去學校，在學校上課。午餐在學校旁邊的餐廳吃飯。下課後在圖書館學習。晚上在家休息。我家在地鐵站前面，非常方便。

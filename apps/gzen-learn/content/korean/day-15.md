## 文法

### 幾點幾分——時鐘的說法

韓語說時間時：
- **小時** → 固有韓語數字 + 시
- **分鐘** → 漢字數字 + 분

| 中文 | 韓語 | 分解 |
|------|------|------|
| 一點 | 한 시 | 固有1 + 시 |
| 兩點 | 두 시 | 固有2 + 시 |
| 三點 | 세 시 | 固有3 + 시 |
| 十二點 | 열두 시 | 固有12 + 시 |
| 三十分 | 삼십 분 | 漢字30 + 분 |
| 一點三十分 | 한 시 삼십 분 | — |
| 兩點十五分 | 두 시 십오 분 | — |

**特殊說法：**
- 半（30分）：반（ban）→ 세 시 반（三點半）
- 幾點？：몇 시예요?
- 幾分？：몇 분이에요?
- 上午：오전（午前）
- 下午：오후（午後）

### 星期幾

韓語的星期名稱來自漢字，和中文邏輯完全相同！

| 星期 | 韓語 | 羅馬拼音 | 漢字原意 |
|------|------|---------|---------|
| 星期一 | 월요일 | wol-yo-il | 月曜日 |
| 星期二 | 화요일 | hwa-yo-il | 火曜日 |
| 星期三 | 수요일 | su-yo-il | 水曜日 |
| 星期四 | 목요일 | mok-yo-il | 木曜日 |
| 星期五 | 금요일 | geum-yo-il | 金曜日 |
| 星期六 | 토요일 | to-yo-il | 土曜日 |
| 星期日 | 일요일 | il-yo-il | 日曜日 |

> **💡 台灣人優勢：** 月火水木金土日！和日文星期相同，台灣人看到就懂！

### 月份和日期

| 月份 | 韓語（漢字數字+월）| 例子 |
|------|-----------------|------|
| 一月 | 일월 | 일월 일일 (1月1日) |
| 二月 | 이월 | 이월 십사일 (2月14日) |
| 三月 | 삼월 | 삼월 일일 (3月1日) |
| 十二月 | 십이월 | 십이월 삼십일일 (12月31日) |

日期說法：**숫자 + 일**（漢字數字）
- 오늘은 몇 월 며칠이에요? （今天幾月幾日？）
- 오늘은 삼월 십오일이에요. （今天是3月15日。）

## 例句

1. **問時間**
   지금 몇 시예요?
   오후 두 시 삼십 분이에요.
   *現在幾點？*
   *下午兩點三十分。*

2. **約定時間**
   몇 시에 만날까요?
   오전 열 시에 만나요.
   *幾點見面呢？*
   *上午十點見面。*

3. **問今天是星期幾**
   오늘 무슨 요일이에요?
   수요일이에요.
   *今天是星期幾？*
   *是星期三。*

4. **問日期**
   오늘은 몇 월 며칠이에요?
   오월 이십일일이에요.
   *今天幾月幾日？*
   *五月二十一日。*

5. **說生日**
   생일이 언제예요?
   십일월 삼일이에요.
   *生日是什麼時候？*
   *十一月三日。*

## 閱讀練習

오늘은 금요일이에요. 오월 이십일이에요. 오전 아홉 시에 수업이 시작해요. 점심은 열두 시에 먹어요. 오후 두 시 반에 한국어 수업이 있어요. 수업이 네 시에 끝나요. 다음 주 월요일에 시험이 있어요. 주말에 열심히 공부할 거예요.

**翻譯：**
今天是星期五，五月二十日。上午九點開始上課，十二點吃午餐。下午兩點半有韓語課，四點下課。下週一有考試，週末要認真學習。

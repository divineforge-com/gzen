## 2024-05-24 - Dropdown Menu Roles Accessibility\n**Learning:** Using `role="menu"` and `role="menuitem"` on a CSS-driven hover dropdown without implementing arrow-key JavaScript navigation traps screen readers in an unexpected interaction model. They expect a rich desktop-like menu but only get standard links.\n**Action:** For simple dropdowns that just contain navigational links, remove ARIA menu roles and rely on standard semantic HTML navigation landmarks to provide a smoother, predictable experience.
